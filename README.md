# froggychips.xyz

Source for [froggychips.xyz](https://froggychips.xyz) — a one-page card.
Plain HTML + CSS, no build step, no JS. Served via GitHub Pages from `main`.

```
index.html   markup
style.css    light + dark, ~120 LOC
CNAME        custom domain (froggychips.xyz)
.nojekyll    skip Jekyll preprocessing
```

To rebuild: edit, commit, push to `main`. Pages auto-deploys.
